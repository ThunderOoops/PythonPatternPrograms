**Topics covered:**
- Triangle Pattern (without built-in functions)
- Pyramid Pattern (without built-in functions)
- Squares based on user input
- Cubes of Odd Numbers from a list
- Diamond Pattern with Alphabets

**Task implementation:**
- `triangle_pattern.py` — Prints a right-angled triangle of `*` using nested while loops, no built-in functions used.
- `pyramid_pattern.py` — Prints a centered pyramid of `*` with proper spacing, using only while loops.
- `squares.py` — Takes numbers as input from the user and prints the square of each number.
- `cubes_odd.py` — Accepts a list of numbers from user, filters only odd numbers, and prints their cubes.
- `diamond_alphabets.py` — Prints a full diamond shape using alphabets (A, B, C...) that go up to the middle and back, using while loops.
