# Python Pattern Programs 🐍

A collection of Python programs covering patterns and number operations — all without using built-in functions.

## 📂 Contents

| File | Description |
|------|-------------|
| `triangle_pattern.py` | Right-angled triangle pattern using `*` |
| `pyramid_pattern.py` | Centered pyramid pattern using `*` |
| `squares.py` | Accepts numbers from user and prints their squares |
| `cubes_odd.py` | Converts a list — cubes only the odd numbers |
| `diamond_alphabets.py` | Diamond shape pattern using alphabets A, B, C... |

## 🛠️ Language
Python 3
